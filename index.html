restful_acl by mdarby

A Ruby on Rails plugin that provides fine grained access control to RESTful resources in a Ruby on Rails 2.0+ project.

Authors

Contact

Download

You can install the gem by running: 

        $ gem sources -a http://gems.github.com
        $ sudo gem install mdarby-restful_acl

You can install the plugin by running: 

$ script/plugin install git://github.com/mdarby/restful_acl

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mdarby/restful_acl

get the source code on GitHub : mdarby/restful_acl
